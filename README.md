# Python Matplotlib Visualizations

A collection of Jupyter notebooks demonstrating data visualization using Python's Matplotlib library.

## Overview

This repository contains beginner-friendly examples and real-world use cases for creating various types of charts and plots with Matplotlib.

## Contents

### Basics
- **1-Intro.ipynb** - Introduction to Matplotlib fundamentals
- **2-line_plot.ipynb** - Line plot examples and techniques
- **3-bar_chart.ipynb** - Bar chart creation and customization

### Real World Examples
- **boxplot.ipynb** - Box plot visualizations
- **histogram.ipynb** - Histogram examples
- **csv-data-visualize.ipynb** - Visualizing data from CSV files
- **fifa-pie-chart.ipynb** - Pie chart using FIFA data
- **gas-line-graph.ipynb** - Line graphs with gas price data

## Datasets

The repository includes sample datasets:
- `fifa_data.csv` - FIFA statistics
- `gas_prices.csv` - Historical gas price data

## Getting Started

1. Install required packages:
   ```bash
   pip install matplotlib pandas jupyter
   ```

2. Open any notebook in Jupyter and run the cells to see visualizations.

## Requirements

- Python 3.x
- Matplotlib
- Pandas (for data manipulation)
- Jupyter Notebook
